# hawk.types

Types for Hawk's modules

## Installation

```
yarn add @hawk.so/types
```

## How to add new types

1. Add new types to `src` folder.
2. Create a pull request.

github-actions bot will check lint styles, build types and commit changes.

### Lint

```
$ yarn lint
```

### Build

```
$ yarn tsc
```
